Dados de Sistemas de Nave:
Dados coletados de uma varredura detalhada de uma nave Thargoid usando um Xeno scanner. Esses dados contêm informações relacionadas aos sistemas internos de uma nave Thargoid. Atualmente, pesquisas com esses dados estão em andamento, com muitos engenheiros especulando que a informação pode ser a chave para o desenvolvimento de estratégias anti-Thargoid e melhorar a tecnologia humana para combater a ameaça representada por essa raça. Portanto, os dados relacionados aos Thargoids são muito procurados para pesquisa científica ou podem ser usados na síntese de vários tipos de munição anti-Thargoid.
